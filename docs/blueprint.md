# **App Name**: Eternal Flame

## Core Features:

- Secret Code Authentication: Require a secret code upon app launch to ensure privacy and exclusivity.
- Main Screen Content Display: Display heartfelt messages, photos, and memories in an engaging format upon successful authentication.

## Style Guidelines:

- Primary color: Deep crimson (#8B0000) to symbolize passion and love.
- Background color: Soft blush (#FDDCDC), a very light tint of crimson, to evoke warmth and gentleness.
- Accent color: Rose gold (#B76E79), a slightly analogous hue, for highlighting interactive elements and creating a sense of luxury.
- Headline font: 'Belleza' sans-serif, for headlines and shorter content. Body font: 'Alegreya' serif, for longer or more sentimental messages.
- Use elegant, minimalist icons that reflect themes of love, memories, and special moments.
- Emphasize full-screen, high-quality imagery with thoughtful content overlays.
- Subtle, smooth transitions and animations to enhance the emotional impact.
